### Brand Colors (Design Tokens)

| Token | Hex | Usage |
|-------|-----|-------|
| `--color-black` | `#000000` | Primary background |
| `--color-dark` | `#0A0A0A` | Alternate section bg |
| `--color-surface` | `#111111` | Cards, elevated surfaces |
| `--color-red` | `#E8000D` | Primary accent (ICG Red) |
| `--color-red-light` | `#FF2D2D` | Hover states |

| `--color-gray-300` | `#CCCCCC` | Body text |
| `--color-gray-400` | `#999999` | Secondary text |
| `--color-gray-600` | `#555555` | Muted/tertiary text |

### Design Features to Replicate in Wix

1. **Sticky navigation** with blur backdrop on scroll
2. **Scroll-triggered animations** (fade up on enter)
3. **Red accent line** on left of service blocks
4. **Gradient glow** effect on hero (subtle red radial)
5. **Grid-based layout** throughout
6. **60/40 ratio bar** on the About > System section
7. **Numbered sections** (01, 02, 03) for visual hierarchy
8. **Hover effects** on cards (border highlight + slight lift)
9. **Qualifying contact form** with dropdowns for service, timeline, budget
