The 5-Layer Production Architecture

Every project runs through the same system — but the agents, tools, and creative decisions change based on what the content demands. Here's exactly how this video was built.

Layer 01

Strategic — Research & Brief

  • Apify agents scraped testimonials from multiple review platforms
  • Perplexity analyzed Nalu's positioning and competitive landscape
  • Top 5 testimonials synthesized into a creative brief
  • Source material: client-provided photos + scraped reviews
Layer 02

Coordination — Orchestration

  • Routes tasks across all specialist agents and APIs
  • Handles parallel execution — video, music, and voiceover generated simultaneously
  • Manages dependencies between asset generation steps
  • Tracks cost and quality across the entire pipeline
Layer 03

Specialist Orchestrators — The Production Floor

  • Video: 24 clips generated via Kling 2.5 from client photos. 92% static shots to reinforce meditative calm. Simulated camera movement on 8% of shots (opening aerials only)
  • Music: Suno V5 composed "Peaceful Awakening" — calm, continuous, 60–80 BPM, ambient/acoustic. 2 variations generated, best selected
  • Voiceover: ElevenLabs produced professional VO from a synthesized testimonial script with emotion tagging for natural delivery
  • Image: Nano Banana generated the end screen and signage elements
Layer 04

Verification — Quality Control

  • Gemini 2.5 Flash + Pro analyzed every single generated clip
  • 24 individual clip analysis JSONs: visual description, location type, key elements
  • Content-based clip mapping — Gemini matched clip content to voiceover sections (not sequential)
  • Misalignment reports generated and corrected across 3 iterative cuts
  • Total QC cost: ~$0.08
Layer 05

Assembly — Final Composition

  • FFmpeg-based composition — no Premiere, After Effects, or Pro Tools at any stage
  • 5+ iterative rough cuts refined through the verification loop
  • Sound design and color correction from the creative brief's storyboard
  • Final delivery: 4K master + 1080p versions

The Beat-Marker System

This is where human direction meets machine execution. The producer places markers over the audio track — defining exactly where each visual cut should land. The system generates a JSON file with frame-accurate timestamps. Agents handle the execution. Humans handle the creative decisions.

25 markers across 89 seconds of audio. Each marker is a creative decision — when to cut, when to hold, when to breathe. The agents execute with frame-level precision, but the rhythm comes from a human ear.

Editing Principles Derived from This Project

  • Rhythm is King — 96% of scenes are exactly 3 seconds. Consistent pacing creates a meditative feel.
  • Hard Cuts Only — 92% hard cuts. Single fade to black before the end card. No transitions for the sake of transitions.
  • Voiceover Leads — Cuts align with natural speech pauses, not music beats. The voice drives the edit.
  • Stillness Creates Calm — 92% static shots. Camera movement reserved for opening aerials only.
  • Visual Arc — Wide (aerial) → Intimate (interior) → Wide (night aerial) → CTA. A journey inward and back out.
  • Uniform Pacing — Slow throughout. No acceleration. The retreat doesn't rush and neither does the video.
